## How to run the app

- Just run `$ docker-compose up` and the app will be running on the port [**`:8000`**](localhost:8000).

## Development

- Go to `~/web/` and run:
  - `$ nvm install`
  - `$ npm i`
  - `$ npm start` and the app will be running at port **`:8080`**.
